Previous work built using Unity3D

Jul 19 2021

In this page is a selection of games that I worked in as personal projects or university assignments.

Convicted#

3D reimagining of Space Invaders as a first-person shooter where you fly a robot in space and destroy aliens using a laser weapon. Made in 2018 as a group class assignment.

Developed for Windows, download link .

Okan#

Rhythm game intended to be a learning tool to help train student nurses on the correct rhythm for the cardiopulmonary resuscitation technique. Has an online API that stores users’ scores and computes a ranking. Made in 2019 as part of the Games, Health and Education course.

Developed for Android, available at the Play Store .

Hollow Word#

Word puzzle game, made for the first miziziz’ game jam in a noncontiguous 48 hour period. Made in 2020 by myself and my younger brother.

Developed for the web, can be played at itch .

Sandbox Simulation#

Work in progress falling sand simulation, inspired by the game Noita. Currently in development and being used as a sample workload to compare Unity’s performance using the Mono and il2cpp backends for my final paper.
